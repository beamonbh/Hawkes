Welcome to the Hawkes Learning Systems Meeting Optimizer!

Here you will be able to create, view and manage your meetings in an efficient manor. The goals of this project is to create a web aplication that allows meeting facilitators to have a tool that streamlines the meeting process as well as allows the "proctor" to view topics they're planning to discuss in a cohesive environment.

If you would like to join, please click the button below. You will be redirected to a page where you will enter the following:

  1. Email Address
  2. First Name
  3. Last Name
